# PredictMatch

PredictMatch es una aplicación web de analítica predictiva enfocada en partidos de fútbol.

## Objetivo
Mostrar partidos, analizar estadísticas recientes de dos equipos y generar una predicción básica del resultado con explicación de factores clave.

## Versión actual
Versión 1.0 en construcción.

## Stack base
- Python
- FastAPI
- pandas
- scikit-learn
- football-data.org
- React

## Características planeadas para la 1.0
- Listado de ligas
- Listado de partidos
- Análisis de partido
- Predicción de local / empate / visita
- Explicación de variables clave
- Interfaz responsive

## Estructura base
- `app/`: backend
- `frontend/`: interfaz web
- `IMG/`: imágenes y logo
- `tests/`: pruebas
- `notebooks/`: experimentos y análisis

Proyecto en fase inicial de configuración.
